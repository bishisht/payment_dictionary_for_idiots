\section*{Common Payment Industry Abbreviations}

Terms that appear frequently in payment contexts:

\begin{description}[leftmargin=3cm, style=nextline]
    \item[ACH] Automated Clearing House
    \item[API] Application Programming Interface
    \item[ATM] Automated Teller Machine
    \item[BIN] Bank Identification Number
    \item[CVV] Card Verification Value
    \item[EMV] Europay, Mastercard, and Visa
    \item[NFC] Near Field Communication
    \item[PAN] Primary Account Number
    \item[PCI DSS] Payment Card Industry Data Security Standard
    \item[POS] Point of Sale
\end{description}

Note: Each of these abbreviations is defined in detail within the main dictionary entries.
